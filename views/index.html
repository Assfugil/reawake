Reawake!

Keep your glitch sites online! (without pinging)

Please enter project invite link or invite code. It may take up to 10 seconds to add or remove a project.

Note: due to the 1000 hours/month limit, you can only keep 1 project up 24/7.

Alternatively, run the follwing command in the glitch console to add your project:

curl -sX PUT https://GlitchUptime.nickchan4.repl.co/api/projects/$PROJECT_INVITE_TOKEN

To remove your project, run the following command:

curl -sX DELETE https://GlitchUptime.nickchan4.repl.co/api/projects/$PROJECT_INVITE_TOKEN
